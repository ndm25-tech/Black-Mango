# Black-Mango

A simple project that implements the "jhi" functionality.

## Usage

To run the jhi script:

```bash
python3 jhi.py
```

This will output: `jhi`

## Testing

To run the tests:

```bash
python3 test_jhi.py
```

## Files

- `jhi.py` - Main script that outputs "jhi"
- `test_jhi.py` - Test script to validate functionality
- `README.md` - This documentation
